## Release trains

Telescope publishes three independent version streams:

| Stream | Tag format | Artifact |
|--------|------------|----------|
| VS Code extension | `extension/vX.Y.Z` | VSIX (Marketplace + OpenVSX) |
| Go module | `server/vX.Y.Z` | `github.com/sailpoint-oss/telescope/server` |
| npm SDK | `sdk/vX.Y.Z` | `@sailpoint-oss/telescope` |

See [docs/PUBLISHING.md](docs/PUBLISHING.md) for release workflow details. GitHub Releases attach VSIX files for manual download at `extension/v*` tags.
